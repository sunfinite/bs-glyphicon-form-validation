Help text will be replaced by the error message if any
Usage

Add the appropriate class to the input field:

<input class="form-control input-md ip" type="text">

Available classes:

  • text: Cannot contain special characters
  • ip: List of comma-separated ips
  • email: List of comma-separated email ids
  • int: Positive integer
On form submit, the first invalid field, if any, gets focus.
